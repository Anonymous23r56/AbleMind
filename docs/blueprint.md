# **App Name**: AbleMind

## Core Features:

- Personalized Onboarding: Allow users to select their AI usage context to personalize the test's tone and task types.
- Dynamic Micro-Challenge Generation: Leverage AI to generate unique micro-challenges for each user, assessing their cognitive skills.
- Behavioral Data Tracking: Monitor and record user behavior (time spent, hesitation) during each task to provide deeper insights into cognitive processes.
- AI-Powered Response Interpretation: Use AI to interpret user responses and behavioral data, providing nuanced insights into their cognitive strengths and weaknesses. An AI agent serves as a tool for this purpose.
- Real-Time Feedback Loop: Dynamically adjust the difficulty and type of micro-challenges based on user performance, ensuring an engaging and relevant testing experience.
- Human-AI Balance Report: Provide users with a visual report summarizing their Human-AI Balance Score, offering personalized feedback and suggestions for improvement.
- Session Management: Handle user sessions by storing all behavioral information to the backend in a database, which facilitates the user balance calculation.

## Style Guidelines:

- Primary color: Electric Blue (#2563EB) to convey intelligence, clarity, and modernity.
- Background color: Light Gray (#F9FAFB) for a clean, soft, and non-distracting backdrop.
- Accent color: Golden Yellow (#FACC15) to highlight key interactive elements and progress indicators, adding warmth and focus.
- Font pairing: 'Inter' for body text, a sans-serif font providing a clean and neutral appearance; 'Space Grotesk' for headlines, lending a techy feel.
- Minimalist design with ample whitespace to reduce cognitive load and maintain a calm, focused user experience, inspired by Apple Human Interface and Notion.
- Subtle fade-in/out transitions and micro-haptics to enhance interactivity and provide gentle feedback to user actions.
- Use clear, modern icons for navigation and to represent different cognitive categories, maintaining a consistent visual language.
